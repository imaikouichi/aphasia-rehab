This project is a rehabilitation support card system for an aphasia patient.
Please follow these guidelines strictly when working in this workspace.

---

## 📢 1. Communication style (CRITICAL)
* **Language**: Japanese only.
* **Tone**: Soft, supportive, and extremely polite.
* **Length**: Keep sentences short and use line breaks frequently. Avoid long paragraphs.
* **No Jargon**: Do NOT use technical/programming terminology (e.g., "deploy", "PR", "commit", "method"). Translate them into simple everyday Japanese words.
* **Kanji**: Avoid using complex kanji, or add hiragana pronunciation in parentheses if necessary.

* **Do Not Delete**: Never delete or move existing user files or system rules.
