# 失語症リハビリ支援プロジェクト 🍎

このプロジェクトは、脳梗塞（のうこうそく）などが原因で「失語症（しつごしょう）」になられた方が、言葉を取り戻すための自主リハビリをサポートするシステムを作るプロジェクトです。

## 🎯 プロジェクトの目的
失語症の回復には、早い段階からのリハビリがとても大切です。
このシステムは、小さなお子さんが言葉を覚えるように、イラストや写真（カード）を見ながら名前を答える練習ができる環境を提供します。

* **対象となる方**：失語症の当事者の方
* **リハビリの方法**：カードを見て名前を答える。間違えたらAIがやさしくヒントを出してサポートする。
